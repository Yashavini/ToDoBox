# ToDoBox

ToDoBox is a simple, chat-style task manager web app where users can add tasks as messages and track their completion using interactive checkboxes. Its interface is inspired by chat apps, making task management engaging and intuitive.

Features

Add tasks in a chat-like input box.

Interactive checkboxes to mark tasks as completed.

Strike-through effect for completed tasks.

Simple, clean, and responsive design.

Easy to use and beginner-friendly.


Demo

You can see the live demo here https://yashavini.github.io/ToDoBox/

Usage

1. Type your task in the input box at the bottom.


2. Press Enter or click the send button.


3. Your task will appear as a chat bubble with a checkbox.


4. Tick/untick the checkbox to mark the task as complete or incomplete.
